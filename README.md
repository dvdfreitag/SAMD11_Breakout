# SAMD11_Breakout
A breakout board for the Atmel ATSAMD11 microcontroller in QFN24

The ATSAMD11 is an extremely inexpensive USB-enabled microcontroller with an Arm Cortex-M0+ core running at a maximum 48MHz.
